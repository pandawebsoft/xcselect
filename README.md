# README.md

## Usage
```
$ xcselect <xcode_version> [--version]
```

ex.
```
    $ xcselect 9.4.1
    $ xcselect 8.2.1
    $ xcselect 7.3.1
```

## Installation
```
git clone https://github.com/pandawebsoft/xcselect.git --depth 1 ./tmp \
&& ./tmp/tools/install && rm -rf tmp
```

## Uninstallation
```
git clone https://github.com/pandawebsoft/xcselect.git --depth 1 ./tmp \
&& ./tmp/tools/uninstall && rm -rf tmp
```
